<a name="introduccion"></a>
## Introducción

La página de inicio, es la página de bienvenida al usuario cada vez que éste inicia sesión en el sistema.
En esta página, se presenta un dashboard informativo sobre las imágenes que se han registrado en el sistema. A continuación, se explica cada sección del dashboard.

<a name="conteo-imagenes"></a>
## Conteo de imágenes

En esta sección, se realiza un conteo de imágenes según diversas características:

<br>

<il>
    <li>
        Imágenes activas.
    </li>
    <li>
        Imágenes etiquetadas.
    </li>
    <li>
        Imágenes por etiquetar.
    </li>
    <li>
        Imágenes eliminadas.
    </li>
</il>

<br>

Para actualizar el conteo de imágenes, haga clic en
<span class="text-green-700 ml-2">
    <i class="fa fa-refresh"></i> Refrescar
</span>.

<a name="incremento-mensual"></a>
## Incremento mensual de imágenes

En esta sección, se muestra un gráfico para mostrar la tendencia de registro de imágenes con respecto a los meses.

<br>

<il>
    <li>
        Se muestra un conteo del total de imágenes registradas en el mes actual, así como en el mes anterior.
    </li>
    <li>
        Usted puede configurar la fecha de inicio y fin de consulta haciendo clic en las fechas abreviadas.
    </li>
</il>

<br>

Para consultar el gráfico según las fechas indicadas, haga clic en
<span class="border rounded-lg p-2 bg-dark text-white">
    Consultar
</span>.


<a name="distribucion-imagenes"></a>
## Distribución de imágenes por etiqueta

En esta sección, se muestra un gráfico que representa la cantidad de imágenes existentes por etiqueta registrada en el sistema.

Además del gráfico, se enlistan las etiquetas registradas, así como el porcentaje de imágenes existentes con respecto al total de imágenes en este repositorio.

Para actualizar el gráfico, haga clic en
<span class="text-green-700 ml-2">
    <i class="fa fa-refresh"></i> Refrescar
</span>.
